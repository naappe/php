# Magnetic Field Viewer

Native Android app that reads `Sensor.TYPE_MAGNETIC_FIELD` and displays genuine X/Y/Z and total field values in µT, a live graph, baseline calibration, and magnetic-change detection.

The app requests no internet, account, camera, microphone, or location permissions. Readings remain on the phone.

Build with Android Studio, or push the project to GitHub and download the `MagneticFieldViewer-debug` artifact from the **Build Android APK** workflow.
